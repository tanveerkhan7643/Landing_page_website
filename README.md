# Landing Page

This is a responsive landing page created using HTML, tailwindcss and JavaScript.  
The project is designed to practice front-end layout, UI design and basic interactivity.

## Features
- Clean and modern UI
- Responsive design (mobile & desktop friendly)
- Smooth scrolling
- Well-structured layout

## Tech Stack
- HTML
- tailwindcss
- JavaScript

## Project Purpose
This project is built for learning and practicing front-end web development concepts such as:
- Page layout
- Styling
- User interaction

## How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser
